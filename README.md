# 🐍 Python Training Repository

Repositório criado com o objetivo de **treinar Python**, consolidar fundamentos da linguagem e **registrar a evolução** de cada participante ao longo do tempo.  
Este projeto serve tanto como material de estudo quanto como histórico de aprendizado.

## 🎯 Objetivo

- Aprender Python do **básico ao intermediário/avançado**
- Praticar conceitos fundamentais da linguagem
- Organizar conteúdos por tópicos (tipos primitivos, estruturas de dados, etc.)
- Permitir que cada participante tenha seu próprio espaço para testes, exercícios e projetos
- Criar o hábito de versionamento com Git/GitHub

## 📂 Organização do Repositório

O repositório é dividido em duas partes principais.

### Conteúdo Geral (Base de Aprendizado)

Pastas organizadas por tema, acessíveis a todos, contendo exemplos, exercícios e estudos sobre Python, como:

- Tipos primitivos
- Estruturas de dados (listas, dicionários, tuplas, sets)
- Condicionais
- Laços de repetição
- Funções
- Classes e Orientação a Objetos
- Outros tópicos conforme o avanço dos estudos

Essas pastas funcionam como uma base comum de aprendizado.

### Pastas Individuais

Cada participante possui sua própria pasta, por exemplo:

/tevo
/breno
/jao  

Dentro delas, cada um pode organizar livremente seus arquivos, incluindo exercícios pessoais, testes de código, desafios, anotações e mini projetos.

## 🛠️ Tecnologias Utilizadas

- Python 3
- Git
- GitHub

## 📈 Evolução do Projeto

Este repositório está em constante evolução.  
Novos conteúdos, exercícios e melhorias serão adicionados conforme o aprendizado avança.

O foco é **aprender e evoluir**, não a perfeição.

## 📌 Observações

- Este é um repositório de estudo
- Errar faz parte do processo
- Código simples é bem-vindo
- Organização e clareza são incentivadas
- Aprendizado colaborativo acima de tudo
